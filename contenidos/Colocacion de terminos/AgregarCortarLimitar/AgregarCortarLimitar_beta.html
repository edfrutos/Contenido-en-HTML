  • Centrar el contenido de una página HTML, puedes usar el siguiente código CSS:

    ×

    Fragmento de código sin formato

    
      body {
        display: flex;
        align-items: center;
        justify-content: center;
        min-height: 100vh; /* esto hace que el contenido ocupe toda la altura de la pantalla */
      }

    • La propiedad "display: flex" indica que los elementos dentro del "body" deben ser alineados usando flexbox. La propiedad "align-items: center" indica que los elementos deben ser alineados verticalmente en el centro, mientras que la propiedad "justify-content: center" indica que los elementos deben ser alineados horizontalmente en el centro. La propiedad "min-height: 100vh" hace que el contenido ocupe toda la altura de la pantalla. Recuerda que debes usar el css en un archivo externo o en la etiqueta style dentro de tu head.

  • Para acortar un párrafo a un número específico de caracteres, creando nuevas líneas y justificando las líneas sin cortar palabras, puedes utilizar JavaScript o un lenguaje de programación en el lado del servidor para dividir el texto y aplicar el formateo deseado. Aquí te doy un ejemplo de cómo podrías hacerlo utilizando JavaScript:

    ×

    Fragmento de código sin formato

    
      
            et paragraph = "Este es un párrafo de ejemplo que será acortado a 100 caracteres. Se asegurará de no cortar palabras y justificar el texto.";

            // Dividir el párrafo en un arreglo de palabras
            let words = paragraph.split(" ");
            
            // Inicializar una variable para almacenar el texto acortado
            let shortenedText = "";
            
            // Inicializar una variable para contar el número de caracteres
            let characterCount = 0;
            
            // Recorrer cada palabra del arreglo
            for (let i = 0; i < words.length; i++) {
                // Añadir la palabra al texto acortado
                shortenedText += words[i] + " ";
                // Aumentar el contador de caracteres
                characterCount += words[i].length + 1;
                if (characterCount >= 100) {
                    // Si se ha alcanzado el límite de caracteres, agregar un salto de línea y resetear el contador
                    shortenedText += "\n";
                    characterCount = 0;
                }
            }
            console.log(shortenedText);
