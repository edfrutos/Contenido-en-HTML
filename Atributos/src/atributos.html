Etiquetas y Atributos

Etiquetas

    Las etiquetas HTML son marcas utilizadas para dar formato al contenido de una página web. Algunas etiquetas comunes incluyen:

  • Encabezados: <h1> a <h6>
  • Párrafos: <p>
  • Listas: <ul> (lista sin orden) y <ol> (lista ordenada)
  • Enlaces: <a>
  • Imágenes: <img>
  • Tablas: <table>
  • Formularios: <form>

    • Las etiquetas deben de ser utilizadas en conjunto para dar formato al contenido de una página web. Es importante recordar que es necesario cerrar las etiquetas para evitar errores en el formato de la página.

Atributos

Los atributos son características adicionales que se pueden agregar a las etiquetas en un documento HTML para proporcionar información adicional sobre esa etiqueta. Por ejemplo, el atributo "class" se puede utilizar para especificar la clase a la que pertenece un elemento, mientras que el atributo "src" se utiliza para especificar la ubicación de una imagen. Los atributos se especifican dentro de las etiquetas de apertura y se componen de un nombre y un valor, separados por un igual. Ejemplo: <img src="image.jpg" alt="descripcion de la imagen">.

Existen varios atributos comunes en HTML que se utilizan para proporcionar información adicional sobre las etiquetas. Algunos ejemplos incluyen:

  1. class: Especifica la clase a la que pertenece un elemento. Se utiliza para aplicar estilos CSS a un conjunto de elementos con la misma clase.

  2. id: Especifica un identificador único para un elemento. Se utiliza para aplicar estilos CSS a un elemento específico y para acceder a ese elemento mediante JavaScript.

  3. src: Especifica la ubicación de un recurso multimedia, como una imagen o un video.

  4. href: Especifica la dirección URL a la que debe navegar un enlace.

  5. alt: Especifica una descripción alternativa de una imagen, que se mostrará si la imagen no se puede cargar.

  6. title: Especifica un título adicional para un elemento, que se mostrará como una sugerencia al colocar el cursor sobre el elemento.

  7. width y height: Especifica el ancho y alto de un elemento multimedia como una imagen.

  8. style: Especifica estilos CSS para un elemento de forma inline.

  9. target: Especifica cómo se debe abrir un enlace, como en una nueva pestaña o ventana.

  10. role: Especifica el rol de un elemento en una página web.

  11. aria-label: Especifica una etiqueta de descripción para un elemento.

Además de los atributos mencionados anteriormente, existen otros atributos en HTML que se utilizan para proporcionar información adicional sobre las etiquetas. Algunos ejemplos incluyen:

  1. name: Especifica un nombre para un elemento, como un campo de formulario.

  2. value: Especifica el valor de un elemento, como un campo de formulario o una opción de selección.

  3. type: Especifica el tipo de un elemento, como un botón, un campo de texto o una casilla de verificación.

  4. placeholder: Especifica un texto de ejemplo que se mostrará en un campo de formulario antes de que el usuario ingrese información.

  5. maxlength: Especifica la longitud máxima de un campo de formulario.

  6. required: Indica si un campo de formulario es obligatorio o no.

  7. readonly: Indica si un campo de formulario es de solo lectura o no.

  8. disabled: Indica si un elemento está deshabilitado o no.

  9. multiple: Indica si un elemento como una lista desplegable permite seleccionar varios valores.

  10. size: Especifica el número de filas visibles para un elemento como una lista desplegable.

  11. pattern: Especifica un patrón para validar el contenido de un campo de formulario.

  12. min y max: Especifica los valores mínimo y máximo permitidos para un elemento como un campo de número.

  13. datalist: Especifica una lista de opciones para un campo de formulario.

  14. cols y rows: Especifica el número de columnas y filas para un elemento como un área de texto.

  15. lang: Especifica el idioma del contenido de un elemento.

  16. xml:lang: Especifica el idioma del contenido de un elemento en un documento XML.

  17. spellcheck: Indica si se debe habilitar o deshabilitar la revisión ortográfica para un elemento.

  18. tabindex: Especifica el orden de navegación de un elemento mediante la tecla TAB.

  19. accesskey: Especifica un atajo de teclado para un elemento.
